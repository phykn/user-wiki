# Judgment Lens

Use this page to decide what must be protected before choosing an action.

Ask:

1. What did the user need preserved?
2. What would make the result a different task if it disappeared?
3. What evidence or artifact would show that the work is better?
4. What is the smallest next action that protects the answer to those questions?

Failure signals:

- The result is smoother, longer, or more polished, but no longer solves the same problem.
- Ambiguity is described, but no decision, edit, or next action changes.
- A past interpretation of the user outranks the current request or current evidence.
- The answer explains the process instead of improving the output.
